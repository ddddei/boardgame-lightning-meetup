# 청년동 보드게임 번개모임 운영 문서

청년동 보드게임 번개모임 당일 운영용 문서입니다. 기준은 진행자를 포함한 8명, 진행자 겸 플레이어, 2시간 기본 진행입니다. 분위기와 시간이 맞으면 18:00 이후에도 이어갈 수 있습니다.

- [당일 TODO 체크리스트](docs/99_todo_checklist.md): 전날부터 마무리까지 확인할 항목

## 운영 전제

- 참가자: 진행자를 포함해 8명
- 진행자: 룰 설명, 시간 관리, 분위기 조율을 맡고 플레이어로도 참여
- 기본 시간: 2시간
- 연장: 18:00 이후 가능하면 추가 진행
- 진행 방향: 현장 분위기와 시간에 맞춰 일부 게임은 조정될 수 있음

## 운영 우선순위

1. 필수: 가짜예술가
2. 우선 진행: 달무티
3. 메인 또는 피날레 후보: 사보타지
4. 시간 부족 시 제외 후보: 젝스님트

## 당일 빠른 판단

- 가짜예술가는 1판이라도 진행한다.
- 달무티는 우선 진행 게임으로 유지한다.
- 사보타지는 35분 이상 확보되면 시작한다.
- 젝스님트는 시간이 넉넉할 때만 넣는다.
- 종료 10분 전에는 새 라운드를 시작하지 않는다.
