사보타지 빠른 룰 요약

진행자가 빠르게 설명할 수 있는 1페이지 요약입니다. 역할 수 등 세부 수치는 룰북으로 확인합니다.

역할 수, 손패 수, 목적지 거리, 보상과 승점은 실제 룰북 확인 필요.

한눈에 보기

광부금 목적지 카드까지 길을 잇는 쪽입니다.
사보타지정체를 숨기고 길 연결을 방해합니다.
역할 비공개역할 카드는 각자 자기 것만 확인합니다.

세팅

  1. 8명 기준 광부/사보타지 카드 수는 실제 룰북 확인 필요.
  2. 역할 카드를 1장씩 비공개 배부합니다.
  3. 남는 역할 카드는 보이지 않게 치웁니다.
  4. 시작 카드 1장을 테이블 중앙 왼쪽에 둡니다.
  5. 목적지 카드 3장을 오른쪽에 둡니다. 거리와 배치는 실제 룰북 확인 필요.
  6. 금 카드 위치는 아무도 모르게 섞어 둡니다.
  7. 길 카드와 행동 카드를 섞고 손패를 나눕니다. 장수는 실제 룰북 확인 필요.

자기 차례 행동

길 놓기길 카드 1장을 놓아 길을 이어갑니다.
행동 사용행동 카드 1장을 사용합니다.
버리기카드 1장을 버리고 새 카드를 뽑습니다.
  • 세부 제한은 실제 룰북 확인 필요.
  • 차례 끝 손패 보충과 더미 소진 처리는 실제 룰북 확인 필요.

고장 / 수리 카드

  • 고장 카드는 특정 플레이어가 길 카드를 놓지 못하게 막습니다.
  • 수리 카드는 해당 고장 상태를 해제합니다.
  • 고장 종류와 수리 카드 대응은 실제 룰북 확인 필요.
  • 고장 상태는 테이블 위에 잘 보이게 둡니다.

시간 부족 운영

  • 남은 시간이 35분 미만이면 새로 시작하지 않습니다.
  • 이미 시작했다면 1라운드만 진행합니다.
  • 룰 질문은 1분 안에 임시 판정합니다.
  • 종료 10분 전에는 새 라운드를 시작하지 않습니다.

라운드 종료 조건

금까지 연결길이 금 목적지 카드까지 연결되면 광부 쪽이 유리한 조건으로 라운드를 종료합니다. 보상과 승점은 실제 룰북 확인 필요.
진행 불가카드 더미와 손패가 소진되어 더 이상 진행할 수 없으면 사보타지 쪽이 유리한 조건으로 라운드를 종료합니다. 정확한 종료 판정은 실제 룰북 확인 필요.
