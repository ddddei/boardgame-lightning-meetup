시간 부족 시 판단

  • 남은 시간이 35분 미만이면 새로 시작하지 않는다.
  • 이미 시작했다면 1라운드만 진행하고 추가 라운드는 하지 않는다.
  • 룰 질문은 1분 안에 임시 판정한다.
  • 종료 10분 전에는 새 라운드를 시작하지 않는다.

세팅 순서

  1. 시작 카드 1장을 테이블 중앙 왼쪽에 놓는다.
  2. 목적지 카드 3장을 오른쪽에 놓는다. 정확한 거리와 배치는 실제 룰북 확인 필요.
  3. 목적지 카드 중 금 카드 위치는 아무도 모르게 섞어 둔다.
  4. 길 카드와 행동 카드를 섞어 더미를 만든다.

  • 수리 카드는 해당 고장 상태를 해제하는 카드다.
