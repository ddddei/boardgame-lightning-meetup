# 출력물 제작 가이드

이미지나 디자인 파일은 만들지 않는다. 현장에서 손에 들고 바로 볼 종이만 준비한다.

## 가짜예술가 역할 카드

- 수량: 8장
- 구성: 7장에는 카테고리/정답, 1장에는 카테고리/X를 적는다.
- 운영: 8장을 모두 접어 섞은 뒤 진행자를 포함한 8명에게 1장씩 배부한다.
- 용도: 진행자가 플레이어를 겸해도 역할 배정이 무작위로 보이도록 준비
- 메모: 진행자도 자기 카드만 보고 참여한다.

## 투표 메모

- 수량: 1장
- 용도: 가짜예술가 투표, 달무티 순위, 사보타지 의심 흐름을 간단히 기록
- 포함 내용: 이름 8칸, 게임별 짧은 메모칸
- 메모: 점수표처럼 정교하게 만들지 말고 진행자가 놓치지 않을 정도면 충분하다.

## 진행자 큐시트

- 수량: 1부
- 포함 내용: 운영 우선순위, 2시간 기본 타임테이블, 축소 판단, 게임별 30초 멘트
- 꼭 넣을 판단: 가짜예술가 필수, 달무티 우선, 사보타지 메인 또는 피날레 후보, 젝스님트 시간 부족 시 제외
- 메모: 종료 10분 전 새 라운드 금지를 크게 적는다.

## 필요 시 사보타지 역할/룰 요약

- 수량: 1부
- 용도: 사보타지를 진행할 때만 확인
- 포함 내용: 8명 역할 구성, 시작 카드와 목적지 카드 세팅, 고장 카드와 수리 카드, 라운드 종료 조건
- 메모: 역할 정보가 참가자에게 보이지 않게 진행자용으로 둔다.
