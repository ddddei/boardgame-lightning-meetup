나 포함 8명이서 진행하게 됨

나는 진행자의 역활과 플레이어의 역할을 동시에 수행하게 됨

일단 플레이 하게 될 게임은

가짜예술가, 달무티, 젝스님트, 사보타지 이렇게 될 것 같다

가짜예술가는 일단 가볍게 아이스브레이킹의 느낌으로 1~2판 정도 진행,

이어서 달무티 → 젝스님트 → 사보타지 이렇게 진행하게 될 듯

달무티도 3판 정도, 젝스님트도 2판 정도, 마무리 사보타지로 진행

18시 넘어서 까지도 진행될 것 같으니 조금 더 시간을 여유롭게 사용해보자

일단 우리가 뒤풀이 장소와 가장 가까울 듯 하다

그리고 일단 청년동 내부에서 진행하는 인원들도 있음

청년동 내부에서는 나 포함 10개 정도의 모임이 진행될 것 같고,

각 모임의 인원은 6명 + 진행자 1명 이렇게 될 것 같다

일단 2시간 정도를 모임 진행으로 생각하고 있고, 진행 시간이 부족하다면 젝스님트를 포기할 생각

일단 텔레스트레이션, 아발론, 시타델도 같이 보유하고 있어서 필요한 게임은 언제든지 진행 가능

일단 가짜 예술가는 필수로 아이스브레이킹으로 진행하게 될 것 같음

진행자로서 룰을 정확하게 알고 있는건 가짜예술가, 달무티 정도임

오늘 내일 해서 룰 숙지가 필요한 상태임

일단 오늘은 번개모임 기획을 어느정도 마무리, 홈페이지에 가볍게 올라갈 썸네일 완성,

필요한 출력물 디자인 가이드 제작 등등 해야할 일들이 있을 것 같음
