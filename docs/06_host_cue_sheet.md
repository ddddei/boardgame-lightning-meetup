# 진행자 큐시트

기준: 나 포함 8명, 진행자 겸 플레이어, 2시간 기본, 18:00 이후 연장 가능. 기본 흐름은 가짜예술가 -> 달무티 -> 젝스님트 -> 사보타지. 부족하면 젝스님트 제외.

## 시작 전 세팅 체크

- [ ] 8명 자리, 필기구, 휴대폰 타이머
- [ ] 가짜예술가 역할 카드 8장, 투표 메모, 큐시트
- [ ] 가짜예술가 카드 8장을 모두 접어 섞고, 진행자를 포함해 1장씩 배부
- [ ] 가짜예술가 먼저 꺼내기, 나머지는 순서대로 정리
- [ ] 사보타지 역할/룰 요약은 진행자만 볼 위치에 두기

## 오프닝 멘트

> 오늘은 2시간 기준으로 가볍게 여러 게임을 해보겠습니다. 저도 같이 플레이해서 세팅, 기록, 카드 정리는 중간중간 나눠 부탁드릴게요. 가짜예술가는 꼭 하고, 달무티를 우선 진행하겠습니다. 시간이 맞으면 사보타지까지 하고, 젝스님트는 여유가 있을 때만 넣겠습니다.

## 운영 시간표

| 상황 | 흐름 | 판단 |
| --- | --- | --- |
| 2시간 기본 | 00:00 안내 -> 00:05 가짜예술가 -> 00:25 달무티 -> 01:05 젝스님트 1판/제외 판단 -> 01:20 사보타지/달무티 -> 01:55 정리 | 사보타지는 35분 이상 남을 때만 |
| 18:00 이후 확장 | 가짜예술가 2판 -> 달무티 3판 이상 -> 젝스님트 1판 -> 사보타지 1라운드 이상 | 2시간 즈음 남을 사람 확인 |
| 축소 운영 | 가짜예술가 1판 -> 달무티 2판 이상 -> 사보타지 1라운드 또는 달무티 추가 | 젝스님트 제외, 설명 길면 달무티 유지 |

## 게임별 설명/전환 멘트

| 게임 | 30초 설명 | 전환 멘트 |
| --- | --- | --- |
| 가짜예술가 | 같은 카테고리 안에서 그림을 한 획씩 이어 그립니다. 한 명만 정답 대신 X를 받은 가짜예술가입니다. 누가 가짜인지 찾고, 가짜는 정답을 맞히면 기회가 있습니다. | 분위기 풀렸으니 달무티로 넘어가겠습니다. |
| 달무티 | 카드를 먼저 다 내려놓으면 이깁니다. 숫자가 낮을수록 강하고, 같은 숫자는 여러 장 낼 수 있습니다. 앞사람보다 강하게 내거나 패스합니다. | 여기서 달무티는 마무리하고 시간 보고 다음 게임을 정하겠습니다. |
| 젝스님트 | 숫자 카드 1장을 동시에 고르고 낮은 숫자부터 줄에 붙입니다. 6번째 카드가 되면 그 줄을 가져가 벌점을 받습니다. 벌점이 적을수록 좋습니다. | 짧게 한 판 했으니 마지막 게임으로 넘어가겠습니다. |
| 사보타지 | 광부 팀은 금까지 길을 잇고, 사보타지는 몰래 방해합니다. 역할은 비밀입니다. 말은 믿어도 되고 의심해도 됩니다. | 여기서 라운드를 마무리하고 구성품을 확인하겠습니다. |

## 진행자 겸 플레이어 주의

- 타이머: 시작, 25분, 60분, 종료 10분 전 확인
- 정보: 가짜예술가 카드는 8장을 모두 접어 섞어 배부하고, 진행자도 자기 카드만 보기
- 분담: 투표, 순위, 카드 섞기는 참가자에게 부탁
- 판정: 룰 질문은 1분 안에 임시 판정
- 참여: 투표/의심 발언 때 한마디씩 기회 주기

## 즉시 판단 기준

| 상황 | 바로 할 일 |
| --- | --- |
| 시간이 부족함 | 젝스님트 제외, 가짜예술가 1판, 달무티 2판, 사보타지는 35분 이상일 때만 |
| 분위기가 처짐 | 가짜예술가 1판 추가 또는 달무티 한 판 더 진행 |
| 룰 설명이 김 | 승리 조건, 차례, 종료 조건만 말하고 시작 |
| 사보타지가 늘어짐 | 새 라운드 금지, 현재 라운드만 끝내고 정리 |

## 마무리 멘트

> 오늘 게임은 여기서 마무리하겠습니다. 카드와 구성품만 같이 확인하고, 18:00 이후 진행이나 뒤풀이 안내를 바로 이어가겠습니다. 좋았던 점이나 아쉬웠던 점도 짧게 알려주세요.
