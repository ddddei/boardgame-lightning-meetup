# 썸네일 디자인 브리프

## 이미지 규격

- 518x298px
- 홈페이지에서 작게 보이는 썸네일 기준
- 문구는 크게, 요소는 적게 배치

## 최종 문구

| 항목 | 문구 |
| --- | --- |
| 메인 | 청년동 보드게임 번개모임 |
| 서브 | 처음 해도 같이 배우며 가볍게 즐겨요 |
| 배지 | 초보 환영 |

## 디자인 톤

- 밝고 편안한 번개모임 분위기
- 보드게임 초보자도 부담 없이 신청할 수 있는 느낌
- 너무 경쟁적이거나 진지하지 않게, 가볍고 친근하게
- 작은 썸네일에서도 메인 문구가 먼저 읽히게 구성

## 반드시 들어갈 시각 요소

- 보드게임 카드 또는 말 2-3개
- 둥근 테이블이나 함께 둘러앉은 느낌을 주는 간단한 요소
- `초보 환영` 배지
- 따뜻한 배경색, 예: 크림색, 연노랑, 연두, 밝은 베이지

## 피해야 할 요소

- 복잡한 게임판이나 작은 텍스트가 많은 구성
- 어두운 배경, 강한 네온색, 과한 대비
- 카지노, 도박, 승부욕이 강해 보이는 이미지
- 특정 게임 박스나 상표가 직접 드러나는 이미지
- 사람이 너무 많거나 표정이 부담스럽게 강조된 이미지
