# 전체 운영 기획

## 목적

진행자를 포함한 8명이 2시간 동안 부담 없이 보드게임을 즐긴다. 가짜예술가로 분위기를 열고, 달무티를 우선 진행한다. 사보타지는 메인 또는 피날레 후보로 두고, 젝스님트는 시간이 부족하면 제외한다.

## 운영 개요

| 항목 | 내용 |
| --- | --- |
| 참가자 | 진행자 포함 8명 |
| 진행자 | facilitator 겸 player |
| 기본 시간 | 2시간 |
| 연장 | 18:00 이후 가능하면 추가 진행 |
| 필수 | 가짜예술가 |
| 우선 진행 | 달무티 |
| 메인 또는 피날레 후보 | 사보타지 |
| 시간 부족 시 제외 후보 | 젝스님트 |

## 운영 원칙

- 현장 분위기와 시간에 맞춰 일부 게임은 조정될 수 있다고 안내한다.
- 시작할 때 2시간 기준이며 진행 순서는 유동적일 수 있다고 안내한다.
- 룰 설명은 승리 조건, 차례 진행, 종료 조건만 먼저 말한다.
- 세팅, 카드 섞기, 점수 기록은 참가자에게 나눠 맡긴다.
- 애매한 룰은 1분 안에 임시 판정하고 다음 판부터 바로잡는다.
- 종료 10분 전에는 새 라운드를 시작하지 않는다.

## 진행자 겸 플레이어 문제와 대응

| 문제 | 상황 | 대응 |
| --- | --- | --- |
| 시간 관리가 흐려짐 | 진행자가 게임에 몰입함 | 휴대폰 타이머를 시작, 25분, 60분, 종료 10분 전 기준으로 맞춘다. |
| 룰 질문이 몰림 | 사보타지나 젝스님트에서 예외가 생김 | 큐시트와 룰 요약을 보고 1분 안에 판정한다. |
| 세팅이 늦어짐 | 설명과 준비를 혼자 처리함 | 한 명은 카드 섞기, 한 명은 점수나 투표 메모를 맡긴다. |
| 진행자가 유리해 보임 | 역할 게임에서 진행자가 정보를 더 아는 것처럼 보임 | 역할 배분은 무작위로 하고, 진행자는 자기 카드만 본다. |
| 설명이 길어짐 | 초보 참가자에게 모든 규칙을 먼저 말하려 함 | 첫 판은 연습판처럼 두고, 카드 효과는 나올 때 설명한다. |
| 조용한 참가자가 빠짐 | 말 많은 사람 중심으로 흘러감 | 가짜예술가 투표나 사보타지 의심 발언 때 한마디씩 돌아가게 한다. |

## 기본 진행 멘트

### 시작 안내

> 오늘은 2시간 기준으로 가볍게 여러 게임을 해보겠습니다. 진행자인 저도 같이 플레이해서, 세팅이나 기록은 중간중간 나눠 부탁드릴게요.

### 게임 선택 안내

> 가짜예술가는 꼭 하고, 달무티를 우선 진행하겠습니다. 시간이 맞으면 사보타지까지 하고, 젝스님트는 여유가 있을 때만 넣겠습니다.

### 시간 조정 안내

> 시간이 조금 빠듯해서 중간 게임은 줄이고 다음 게임으로 넘어가겠습니다.

### 마무리 안내

> 여기서 게임은 마무리하겠습니다. 카드와 구성품만 같이 확인하고, 이후 일정은 바로 안내드릴게요.
