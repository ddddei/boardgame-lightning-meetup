# 타임테이블

## 운영 기준

- 기준 인원은 진행자를 포함한 8명이다.
- 진행자는 플레이어를 겸한다.
- 기본 시간은 2시간이다.
- 18:00 이후 가능하면 확장 진행한다.
- 운영 우선순위는 필수 가짜예술가, 우선 진행 달무티, 메인 또는 피날레 후보 사보타지, 시간 부족 시 제외 후보 젝스님트다.

## 2시간 기본 타임테이블

| 시간 | 진행 | 판단 기준 |
| --- | --- | --- |
| 00:00-00:05 | 인사, 자리 정리, 진행 방식 안내 | 현장 분위기와 시간에 맞춰 일부 게임은 조정될 수 있다고 안내 |
| 00:05-00:25 | 가짜예술가 1판, 가능하면 2판 | 00:25에는 달무티로 이동 |
| 00:25-01:05 | 달무티 2판 이상 | 분위기가 좋으면 1판 추가 |
| 01:05-01:20 | 젝스님트 1판 또는 제외 판단 | 시간이 빠듯하면 제외 |
| 01:20-01:55 | 사보타지 1라운드 또는 달무티 추가 | 사보타지는 35분 이상 남을 때만 시작 |
| 01:55-02:00 | 구성품 확인, 종료 안내 | 새 라운드 금지 |

## 18:00 이후 확장 타임테이블

18:00 이후 시간이 가능하고 참가자 동의가 있을 때만 사용한다.

| 시간 | 진행 | 판단 기준 |
| --- | --- | --- |
| 00:00-00:05 | 인사, 자리 정리, 진행 방식 안내 | 2시간 뒤 계속할지 다시 묻겠다고 안내 |
| 00:05-00:25 | 가짜예술가 2판 | 분위기 풀기 |
| 00:25-01:10 | 달무티 3판 이상 | 우선 진행 게임으로 충분히 진행 |
| 01:10-01:30 | 젝스님트 1판 | 여유가 있을 때만 진행 |
| 01:30-02:20 | 사보타지 1라운드 이상 | 메인 또는 피날레로 진행 |
| 02:20-02:30 | 정리, 뒤풀이 또는 추가 진행 안내 | 남는 사람만 이어가기 |

## 축소 진행

시작이 늦거나 설명이 길어졌을 때 사용한다. 젝스님트는 제외한다.

| 시간 | 진행 | 판단 기준 |
| --- | --- | --- |
| 00:00-00:05 | 인사, 진행 방식 안내 | 바로 첫 게임 시작 |
| 00:05-00:20 | 가짜예술가 1판 | 필수 진행 |
| 00:20-01:05 | 달무티 2판 이상 | 우선 진행 |
| 01:05-01:55 | 사보타지 1라운드 또는 달무티 추가 | 사보타지 설명 시간이 부족하면 달무티 유지 |
| 01:55-02:00 | 정리, 종료 안내 | 구성품 확인 |

## 현장 판단 문장

- 기본 시작: "오늘은 2시간 기준이고, 현장 분위기와 시간에 맞춰 일부 게임은 조정될 수 있습니다."
- 축소 전환: "시간이 빠듯해서 젝스님트는 건너뛰고 다음 게임으로 넘어가겠습니다."
- 확장 전환: "18:00 이후에도 괜찮은 분들은 사보타지까지 이어가겠습니다."
