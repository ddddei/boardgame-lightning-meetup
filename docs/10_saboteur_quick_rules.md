# 사보타지 빠른 룰 요약

진행자 확인용 요약입니다. 정확한 역할 카드 수나 세부 규칙은 실제 보유 룰북 확인 필요.

## 8명 기준 역할 구성 확인란

- [ ] 광부 카드 수 확인: 실제 보유 룰북 확인 필요
- [ ] 사보타지 카드 수 확인: 실제 보유 룰북 확인 필요
- [ ] 8명에게 역할 카드 1장씩 비공개 배부
- [ ] 남는 역할 카드는 보이지 않게 따로 치우기

## 세팅 순서

1. 시작 카드 1장을 테이블 중앙 왼쪽에 놓는다.
2. 목적지 카드 3장을 오른쪽에 놓는다. 정확한 거리와 배치는 실제 보유 룰북 확인 필요.
3. 목적지 카드 중 금 카드 위치는 아무도 모르게 섞어 둔다.
4. 길 카드와 행동 카드를 섞어 더미를 만든다.
5. 각자 시작 손패 수를 나눠준다. 정확한 장수는 실제 보유 룰북 확인 필요.
6. 역할 카드는 각자 자기 것만 확인한다.

## 자기 차례에 할 수 있는 행동

아래 중 하나를 선택한다. 세부 제한은 실제 보유 룰북 확인 필요.

- 길 카드 1장을 놓아 길을 이어간다.
- 행동 카드 1장을 사용한다.
- 카드 1장을 버리고 새 카드를 뽑는다.

차례가 끝나면 손패를 보충한다. 더미가 떨어졌을 때의 처리와 손패 보충 규칙은 실제 보유 룰북 확인 필요.

## 고장/수리 카드

- 고장 카드는 특정 플레이어가 길 카드를 놓지 못하게 막는 카드다.
- 수리 카드는 해당 고장 상태를 해제하는 카드다.
- 고장 종류와 수리 카드의 대응 관계는 실제 보유 룰북 확인 필요.
- 진행자가 플레이어를 겸하므로 고장 상태는 테이블 위에 잘 보이게 두고, 참가자 한 명이 함께 확인한다.

## 라운드 종료 조건

- 길이 금 목적지 카드까지 연결되면 광부 쪽이 유리한 조건으로 라운드를 종료한다. 보상과 승점 처리는 실제 보유 룰북 확인 필요.
- 카드 더미와 손패가 소진되어 더 이상 진행할 수 없으면 사보타지 쪽이 유리한 조건으로 라운드를 종료한다. 정확한 종료 판정은 실제 보유 룰북 확인 필요.
- 목적지 카드 공개 시점과 연결 판정은 실제 보유 룰북 확인 필요.

## 시간 부족 시 운영 기준

- 남은 시간이 35분 미만이면 새로 시작하지 않는다.
- 이미 시작했다면 1라운드만 진행하고 추가 라운드는 하지 않는다.
- 룰 질문이 길어지면 1분 안에 임시 판정하고, 다음 판부터 룰북 기준으로 바로잡는다.
- 종료 10분 전에는 새 라운드를 시작하지 않는다.
